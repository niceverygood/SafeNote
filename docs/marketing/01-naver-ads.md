# SafeNote 네이버 검색광고(파워링크) 카피 세트

대상: 전담 안전관리자가 없는 5~49인 사업장, 비전문가 대표. 후킹 축은 "의무 이행 증빙을 상시 자동 축적 → 사고 시 면책 입증 자료". 톤은 신뢰할 수 있는 advisor(불안→통제), 능동태, 정직하되 비관 금지.

## 4. 광고그룹 구조 제안

키워드를 의도별로 3개 광고그룹으로 분리. 그룹별로 소재 앵글을 맞춰 품질·전환율 관리.

### 그룹 A — "적용·처벌 우려"(불안→통제)
- 키워드: 중대재해처벌법 5인 / 50인 미만 중대재해법 / 중대재해처벌법 50인 미만 적용 / 중대재해 대표 처벌 / 중대재해처벌법 대표이사 / 중대재해 면책
- 추천 제목: 1, 2, 8 / 추천 설명: 1, 3, 4
- 메시지 초점: "증빙이 곧 면책" → 막연한 불안을 점검 가능한 통제로 전환.

### 그룹 B — "체계 구축·준비"(무엇을 해야 하나)
- 키워드: 안전보건관리체계 구축 / 안전보건관리체계 구축 방법 / 중대재해처벌법 대비 / 중대재해처벌법 준비 / 중대재해처벌법 서류 / 안전보건 의무 이행 / 중대재해처벌법 컨설팅 / 소규모 사업장 안전관리 / 안전관리자 없는 사업장
- 추천 제목: 5, 7, 4 / 추천 설명: 1, 6, 8
- 메시지 초점: 전담자 없이도 의무 점검·증빙 축적을 한 곳에서.

### 그룹 C — "도구·진단·서식"(즉시 실행)
- 키워드: 위험성평가 양식 / 위험성평가 작성 방법 / 위험성평가 예시 / 중대재해처벌법 체크리스트 / 중대재해 자가진단
- 추천 제목: 3, 6, 8 / 추천 설명: 2, 5, 7
- 메시지 초점: 가입 없이 5분 무료 진단으로 즉시 시작. (위험성평가 자동화를 메인 후킹으로 쓰지 않고, 진단·증빙 시스템으로 후킹)
